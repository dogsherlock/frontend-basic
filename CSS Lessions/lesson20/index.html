CSS Numerical Units

Absolute (fixed) units

  • cm - centimerters
  • mm - millimeters
  • in - inches
  • px - pixels
  • pt - points
  • pc - picas

Relateive (fluid) units

  • em - rel to container
  • ex - rel to x height of current font
  • ch - rel to the width of 0
  • rem - rel to font-size of root element
  • vw - rel to %1 of viewport width
  • vh - rel to %1 of viewport height
  • vmin - rel to %1 of viewport smaller dimension
  • vmax - rel to %1 of viewport larger dimension
  • % - rel to parent element
